# Software and Web Tools Used

## Command-line tools

- NCBI Datasets
- QUAST
- FastANI
- AMRFinderPlus
- Kleborate
- Panaroo
- IQ-TREE
- Python
- Bash/Linux

## Web-based tools

- SWISS-MODEL: protein structure modelling
- PROCHECK: Ramachandran plot validation
- VaxiJen v2.0: antigenicity prediction
- AllerTOP v2.1: allergenicity prediction
- IEDB B-cell epitope prediction: BepiPred Linear Epitope Prediction 2.0
- IEDB MHC-I binding prediction: NetMHCpan 4.1 EL
- IEDB MHC-II binding prediction: NetMHCIIpan 4.1 EL
- ToxinPred2: toxicity prediction
- IEDB population coverage tool
- GalaxyPepDock: peptide-HLA docking
- PyMOL: structural visualization
